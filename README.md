


# LAMZU Polling Rate Tray

System tray utility to switch polling rate on LAMZU 8K mice.

## Install

```
pip install pystray pillow pywinusb
```

## Usage

Run `polling_tray.pyw`. Right-click the tray icon (1K/4K/8K) to switch.

## Autostart

Create a shortcut in `shell:startup` folder.

## Files

- `polling_tray.pyw` - main program
- `polling_rate.json` - last saved rate
